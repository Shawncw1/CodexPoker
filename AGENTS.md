# CodexPoker Agent Rules

## Project Isolation
- Treat this repository as a clean-slate poker app effort.
- Do not read, import, copy, or reference code/content from other local poker repositories unless the user explicitly asks in that turn.
- Do not reference or reuse prior poker attempts from the user's GitHub unless explicitly requested in that turn.
- Keep all new implementation work for this app inside this repository.

## Allowed External References
- It is allowed to reference public poker resources, documentation, and open-source poker projects on the internet when useful.
